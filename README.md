# ThreatLens

ThreatLens is a lightweight cybersecurity reconnaissance tool designed to help analysts quickly gather intelligence on domains, IPs, and files from a single CLI interface.

Built for simplicity, speed, and real-world SOC workflows.

---

## ⚡ Features

- 🌐 Scan Domains / URLs (urlscan.io integration)
- 🌍 IP Intelligence Lookup
- 📡 HTTP Header Inspection
- 🔐 File SHA256 Hash Generator
- 🧠 Interactive CLI Menu

---

## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/RUNNINDEAD/threatlens.git
cd threatlens

```
